## Codewars Java Übungsprojekt
Überblick

Dieses Repository enthält meine Lösungen zu verschiedenen Codewars Katas in Java.
Der Fokus liegt auf dem systematischen Lernen von Java Grundlagen, sauberer Logik, verständlichem Code und dem Einsatz unterschiedlicher Lösungsansätze.

Alle Aufgaben sind nach kyu Stufen strukturiert und zeigen meinen Fortschritt von einfachen Grundlagen bis zu komplexeren Problemen mit Streams, Maps und Algorithmen.

Inhalte des Projekts

Im Projekt sind Lösungen zu folgenden Themen enthalten:

• Arbeiten mit Arrays und Listen
• String Verarbeitung und Zeichenanalyse
• Bedingungen und Kontrollstrukturen
• Schleifen und verschachtelte Logik
• Nutzung von Collections wie ArrayList und HashMap
• Einsatz von Java Streams und Lambdas
• Schreiben und Verstehen von Unit Tests
• Performance Überlegungen bei größeren Datenmengen

Struktur

Die Aufgaben sind nach kyu Stufen organisiert:

• 8 kyu
Einfache Grundlagenaufgaben zum Einstieg
Beispiel Boolean Ausgaben Palindrome Traffic Lights Plural

• 7 kyu
Erweiterte Logik mit Strings Arrays und Bedingungen
Beispiel Jaden Case Exes and Ohs Remove Anchor Arithmetic Functions

• 6 kyu
Anspruchsvollere Aufgaben mit Algorithmen und Datenstrukturen
Beispiel Array Diff Equal Sides of an Array Find the Unique Number Count Smileys

Jede Kata enthält
die Aufgabenbeschreibung
eine oder mehrere eigene Lösungen
in vielen Fällen zusätzliche Varianten mit Streams oder alternativer Logik
Unit Tests zur Überprüfung der Funktionalität

Lernfortschritt

Durch dieses Projekt habe ich gelernt:

• Probleme strukturiert zu analysieren
• Lösungen schrittweise zu entwickeln
• Code lesbar und nachvollziehbar zu schreiben
• verschiedene Lösungsansätze zu vergleichen
• imperativen Code und funktionalen Stil zu kombinieren
• Java Streams gezielt und sinnvoll einzusetzen
• Tests als festen Bestandteil der Entwicklung zu nutzen

Der Code zeigt bewusst nicht nur die kürzeste Lösung, sondern auch meinen Lernweg und meine Gedankengänge.

Technologien

• Java 17
• JUnit für Tests
• Codewars als Trainingsplattform

Ziel des Projekts

Dieses Repository dient
als Lerndokumentation

und als Referenz für meinen Fortschritt Richtung Softwareentwicklung

Es ist bewusst praxisnah aufgebaut und wird kontinuierlich erweitert.
